# Tranquil

Soft colors for sane person.

![tranquil_logo](https://user-images.githubusercontent.com/101834410/218586952-7f1f2e20-1794-4997-8402-9dd33c0b7ca2.png)

## Installation

```bash
git clone https://github.com/KoBruhh/tranquil-vim.git
cd tranquil-vim
chmod +x INSTALL
./INSTALL --nvim # or --vim
```
